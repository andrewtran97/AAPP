# Responsible Disclosure Checklist

## Scope

- [ ] Written scope exists
- [ ] Target/component is in scope
- [ ] Tool/action type is allowed
- [ ] Evidence was collected inside scope

## Evidence

- [ ] Trace verifies
- [ ] Bundle exists
- [ ] Replay report exists
- [ ] Hashes exist
- [ ] Verification result exists

## Redaction

- [ ] Raw secrets removed
- [ ] Private keys removed
- [ ] Bystander data removed
- [ ] Public material separated
- [ ] Report scanned for secret-like values

## Disclosure

- [ ] Maintainer contact exists
- [ ] Private email prepared
- [ ] Timeline prepared
- [ ] Embargo checklist prepared
- [ ] Publication checklist prepared
