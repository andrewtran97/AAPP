# Legal Naming Checklist

Stop if any public name:

- contains CISA
- contains Qubes
- implies government affiliation
- implies certification not granted
- implies endorsement not granted
- claims post-quantum secure before implementation and review
- claims unbreakable encryption
- uses military-grade as marketing language

Required before public launch:

- trademark search
- domain search
- GitHub organization/repo name review
- package registry name review
- competitor name review
